## Что значит «изменить основу»?

Изменить основу — значит заменить ядро или платформу запуска, предустановленные для услуги в рамках вашего тарифа. Ниже перечислены основы, которые мы можем установить автоматически для каждого типа тарифа.

> **Внимание!**
>
> Список доступных предустановок не ограничивает вас только указанными вариантами. Например, если для сервера предустановлено ядро `Paper`, вы можете самостоятельно установить `Purpur` или `Folia` — сервер продолжит работать.

> **Важно!**
>
> Нестандартные платформы необходимо устанавливать и настраивать самостоятельно. Администрация не помогает с их установкой и настройкой.

## Доступные основы

**«Предустановка недоступна»** означает, что для этого типа тарифа нельзя выбрать другую основу при предустановке: мы не предлагаем альтернативные ядра или платформы для автоматической установки.

| Игра или тип сервера в тарифе | Доступные основы |
| --- | --- |
| **Minecraft** | `Velocity`, `BungeeCord`, `Vanilla`, `Paper`, `Purpur`, `Forge`, `Fabric`, `Bedrock` |
| **MTA** | **Предустановка недоступна** |
| **Terraria** | **Предустановка недоступна** |
| **Веб-сервер** | `HTML/PHP`, `Next.js`, `Vue`, `React` |
| **Скриптовый сервер** | `Python`, `JavaScript` |
