## Що означає «змінити основу»?

Змінити основу — означає замінити ядро або платформу запуску, попередньо встановлені для послуги в межах вашого тарифу. Нижче наведено основи, які ми можемо автоматично встановити для кожного типу тарифу.

> **Увага!**
>
> Список доступних попередніх установок не обмежує вас лише зазначеними варіантами. Наприклад, якщо для сервера попередньо встановлено ядро `Paper`, ви можете самостійно встановити `Purpur` або `Folia` — сервер продовжить працювати.

> **Важливо!**
>
> Нестандартні платформи необхідно встановлювати та налаштовувати самостійно. Адміністрація не допомагає з їх установленням і налаштуванням.

## Доступні основи

**«Попереднє встановлення недоступне»** означає, що для цього типу тарифу не можна вибрати іншу основу під час попереднього встановлення: ми не пропонуємо альтернативні ядра або платформи для автоматичного встановлення.

| Гра або тип сервера в тарифі | Доступні основи |
| --- | --- |
| **Minecraft** | `Velocity`, `BungeeCord`, `Vanilla`, `Paper`, `Purpur`, `Forge`, `Fabric`, `Bedrock` |
| **MTA** | **Попереднє встановлення недоступне** |
| **Terraria** | **Попереднє встановлення недоступне** |
| **Вебсервер** | `HTML/PHP`, `Next.js`, `Vue`, `React` |
| **Скриптовий сервер** | `Python`, `JavaScript` |
