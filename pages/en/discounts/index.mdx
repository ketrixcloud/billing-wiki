We run discounts fairly rarely — that's because we already offer the lowest market prices for this CPU and these specs.

You can find out when a discount is coming in our Discord community, in the [discounts](https://discord.com/channels/1293467162198933505/1524131593504161895) channel.

## Permanent discount for the Discord server tag

We offer a permanent discount for having our server tag equipped in Discord.

![The "Server Tag" item in the Discord server menu](discounts/discord-server-tag-menu.png)

The discount is applied automatically once you've had the tag equipped for more than **a week**. You get a **5%** discount on any service (purchase or renewal) — with no limit on how many times you can use it.

> **Important!** Removing the tag loses you the discount immediately. You can get it back only a week after re-equipping the tag.
