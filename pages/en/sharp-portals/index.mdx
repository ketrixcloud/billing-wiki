## What does “changing the server base” mean?

Changing the server base means replacing the runtime or server platform preinstalled for your service under the selected plan. Below are the bases that we can install automatically for each plan type.

> **Please note!**
>
> The available presets do not limit you to the listed options. For example, if your server comes with `Paper`, you can install `Purpur` or `Folia` yourself, and the server will continue to work.

> **Important!**
>
> Custom platforms must be installed and configured by you. The administration team does not assist with their installation or configuration.

## Available server bases

**“Preinstallation unavailable”** means that you cannot select a different base for this plan during setup: we do not offer alternative runtimes or platforms for automatic installation.

| Game or server type included in the plan | Available server bases |
| --- | --- |
| **Minecraft** | `Velocity`, `BungeeCord`, `Vanilla`, `Paper`, `Purpur`, `Forge`, `Fabric`, `Bedrock` |
| **MTA** | **Preinstallation unavailable** |
| **Terraria** | **Preinstallation unavailable** |
| **Web server** | `HTML/PHP`, `Next.js`, `Vue`, `React` |
| **Script server** | `Python`, `JavaScript` |
